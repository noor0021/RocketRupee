# 🚀 RocketRupee Demo Guide

## Quick Start Flow

### 1. Launch Experience
- **Splash Screen** (3 seconds)
  - Watch the rocket animation
  - Logo reveal with tagline

### 2. Onboarding (30 seconds)
- 4 introductory screens explaining the concept
- Enter your name to create profile
- Click "Start Your Journey"

### 3. Dashboard (Home Base)
- View your XP progress bar
- See level, portfolio value, streak, and badges
- Check today's recommended mission
- Quick access to SIP Simulator and Leaderboard

### 4. Complete Missions
- Tap on "Missions" in bottom navigation
- Select an unlocked mission
- Answer the interactive question
- Earn XP and badges!

### 5. Try the SIP Simulator
- Adjust monthly SIP amount (₹500 - ₹50,000)
- Set investment period (1-30 years)
- Choose expected return rate (1-20%)
- Watch the animated chart show wealth growth

## Troubleshooting

### Data Reset
Clear browser localStorage:
```javascript
localStorage.clear()
```
Then refresh page.

### Mission Already Completed
Missions can only be completed once. Check Profile → Mission History to see completed missions.

### Level Not Increasing
Each level requires 500 XP. Check Dashboard to see progress towards next level.
